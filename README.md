## SimplePhonebook
Copyright © Nusantara Project

###Introduction

SimplePhonebook is a simple phonebook application made with C language. SimplePhonebook
have command line interface. 


###Features

* Add New Contact
* View List Contact
* Search Contact
* Delete Contact
* Save Contact
